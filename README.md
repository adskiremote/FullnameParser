# FullnameParser

### Author: Adskiremote

### Summary
A human fullname parser to determine Title, Firstname, Middlename, Lastname and Suffix.

## Usage:
````
require('fullname-parser.php');
$names = parse_name($fullname);
````

### Output result:
```$names['title'];
$names['firstname'];
$names['middle'];
$names['lastname'];
$names['suffix'];
```
